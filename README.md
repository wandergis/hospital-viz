# hospital-viz
根据网络来源的数据整理出经纬度坐标，生成geojson文件并可视化展示，仅作为可视化展示学习之用，仅供参考学习leaflet。
## 数据来源
- **数据来自网络，对数据的准确性并不完全保证，可视化结果仅供参和考学习用途。**
- [来源一](http://news.ifeng.com/mainland/special/ptxyy/)
- [来源二](https://github.com/langhua9527/Hospital)
